# Настройка формы обратной связи

## Локальная разработка

Для работы формы в режиме разработки нужно запустить локальный сервер:

1. **Установите зависимости** (если еще не установлены):
   ```bash
   npm install
   ```

2. **Создайте файл `.env`** в корне проекта:
   ```env
   TELEGRAM_BOT_TOKEN=8548638828:AAFm1l0cnI2CLy4GDjedC2V37OqiAE_ItzU
   TELEGRAM_CHAT_ID=ваш_chat_id
   PORT=3000
   ```

3. **Запустите локальный сервер** (в отдельном терминале):
   ```bash
   npm run dev:server
   ```

4. **Запустите Vite dev server** (в другом терминале):
   ```bash
   npm run dev
   ```

Теперь форма будет работать локально и отправлять сообщения в Telegram.

## Production (Vercel/Netlify)

В production используйте serverless функцию из папки `api/telegram-send.ts`.

### Для Vercel:
- Функция автоматически развернется из папки `/api`
- Добавьте переменные окружения в настройках проекта:
  - `TELEGRAM_BOT_TOKEN`
  - `TELEGRAM_CHAT_ID`

### Для Netlify:
- Переместите файл в `/netlify/functions/telegram-send.ts`
- Добавьте переменные окружения в настройках сайта
