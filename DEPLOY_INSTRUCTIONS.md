# Инструкции по деплою портфолио на Vercel

## Шаг 1: Получение Telegram Chat ID

Перед деплоем нужно получить ваш Chat ID для настройки формы обратной связи.

### Способ 1: Через бота @userinfobot (самый простой)
1. Откройте Telegram
2. Найдите бота @userinfobot
3. Отправьте ему любое сообщение
4. Бот вернет ваш Chat ID (число вида `123456789`)
5. Скопируйте этот ID - он понадобится на шаге 5

### Способ 2: Через getUpdates API
1. Откройте в браузере:
   ```
   https://api.telegram.org/bot8548638828:AAFm1l0cnI2CLy4GDjedC2V37OqiAE_ItzU/getUpdates
   ```
2. Сначала отправьте сообщение своему боту в Telegram (любое сообщение)
3. Обновите страницу в браузере
4. Найдите в ответе JSON строку: `"chat":{"id":123456789}`
5. Число после `"id":` - это ваш Chat ID

## Шаг 2: Создание GitHub репозитория

### Если Git еще не инициализирован:

1. Откройте терминал в папке проекта
2. Выполните команды:

```bash
git init
git add .
git commit -m "Initial commit: Portfolio website"
```

### Создание репозитория на GitHub:

1. Зайдите на [github.com](https://github.com) и войдите в аккаунт
2. Нажмите кнопку "+" в правом верхнем углу → "New repository"
3. Заполните:
   - Repository name: `portfolio` (или любое другое имя)
   - Description: "Portfolio website for Daria Koval"
   - Выберите Public или Private
   - **НЕ** отмечайте "Initialize this repository with a README" (проект уже готов)
4. Нажмите "Create repository"

### Подключение локального репозитория к GitHub:

1. GitHub покажет инструкции. Выполните в терминале:

```bash
git remote add origin https://github.com/ВАШ_USERNAME/portfolio.git
git branch -M main
git push -u origin main
```

Замените `ВАШ_USERNAME` на ваш GitHub username и `portfolio` на имя вашего репозитория.

## Шаг 3: Деплой на Vercel

### Через GitHub (рекомендуется):

1. Зайдите на [vercel.com](https://vercel.com)
2. Нажмите "Sign Up" и выберите "Continue with GitHub"
3. Разрешите Vercel доступ к вашему GitHub аккаунту
4. После входа нажмите "Add New Project"
5. Найдите ваш репозиторий `portfolio` в списке
6. Нажмите "Import"
7. Vercel автоматически определит настройки:
   - Framework Preset: Vite
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - Install Command: `npm install`
8. **Пока НЕ нажимайте Deploy!** Сначала нужно настроить переменные окружения (следующий шаг)

### Через Vercel CLI (альтернативный способ):

1. Установите Vercel CLI:
   ```bash
   npm install -g vercel
   ```

2. В корне проекта выполните:
   ```bash
   vercel
   ```

3. Следуйте инструкциям:
   - Логин через браузер
   - Подтвердите настройки проекта
   - Дождитесь деплоя

## Шаг 4: Настройка переменных окружения

**Важно:** Это критический шаг для работы формы обратной связи!

1. В настройках проекта Vercel перейдите в раздел **"Settings"**
2. Откройте **"Environment Variables"**
3. Добавьте следующие переменные:

   **Переменная 1:**
   - Name: `TELEGRAM_BOT_TOKEN`
   - Value: `8548638828:AAFm1l0cnI2CLy4GDjedC2V37OqiAE_ItzU`
   - Environment: выберите все (Production, Preview, Development)

   **Переменная 2:**
   - Name: `TELEGRAM_CHAT_ID`
   - Value: `ваш_chat_id` (тот, который вы получили на шаге 1)
   - Environment: выберите все (Production, Preview, Development)

4. Нажмите "Save" для каждой переменной

## Шаг 5: Запуск деплоя

1. После настройки переменных окружения вернитесь на вкладку **"Deployments"**
2. Если деплой уже начался автоматически, дождитесь его завершения
3. Если нет, нажмите "Deploy" или создайте новый деплой
4. Дождитесь завершения сборки (обычно 1-3 минуты)

## Шаг 6: Проверка работоспособности

После успешного деплоя:

1. Откройте URL вашего проекта (будет показан в Vercel Dashboard)
2. Проверьте основные функции:
   - ✅ Все секции сайта отображаются корректно
   - ✅ Навигация работает
   - ✅ Изображения загружаются
   - ✅ Форма обратной связи открывается
3. Протестируйте форму:
   - Заполните все поля
   - Отправьте тестовое сообщение
   - Проверьте Telegram - должно прийти сообщение

## Шаг 7: Настройка кастомного домена (опционально)

Если хотите использовать свой домен или красивое имя:

1. В Vercel Dashboard откройте **Settings → Domains**
2. Добавьте домен:
   - Для бесплатного поддомена Vercel: просто используйте предоставленный URL
   - Для своего домена: введите домен и следуйте инструкциям по настройке DNS

## Устранение проблем

### Форма не отправляется:
1. Проверьте консоль браузера (F12 → Console) на ошибки
2. Проверьте Network tab - должен быть запрос к `/api/telegram-send`
3. Проверьте переменные окружения в Vercel (особенно `TELEGRAM_CHAT_ID`)
4. Проверьте логи функции: Vercel Dashboard → Functions → telegram-send → Logs

### Сборка не проходит:
1. Проверьте логи сборки в Vercel Dashboard
2. Попробуйте собрать локально: `npm run build`
3. Убедитесь, что все зависимости установлены

### API функция не работает:
1. Проверьте, что файл `api/telegram-send.ts` находится в корне проекта
2. Проверьте логи функции в Vercel Dashboard
3. Убедитесь, что переменные окружения правильно настроены
